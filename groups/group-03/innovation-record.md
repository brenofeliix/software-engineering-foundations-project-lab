# Registro de Inovação

## Nome da Solução
Jogando que se aprende 

---

## Problema
Aulas monótonas e evasão escolar

---

## Solução Proposta
Torna as aulas mais interativas e dinâmicas fazendo com que os alunos foquem mais nas aulas e aprendam de diferentes maneiras, facilitando o processo de Aprendizagem

---

## Diferencial Inovador
Esse projeto busca ser implementado nas escolas como uma plataforma de ensino para tornar a Fixação do conteúdo diferente.

---

## Potencial de Aplicação
Em todas as escolas de âmbito nacional, do sexto ao nono ano do fundamental

---

## Tecnologias Utilizadas
- Programação em backend: C# e kotlin
- Programação em fronend: CSS e HTML
- Visual estudio e Git Hub serão utilizados para compilação
- Banco de dados: será utilizado a linguagem XML
- Sera utilizado a API REST para fazer a integração entre banco de dados e o sistema de backend. 
---

## Possível contribuição científica ou tecnológica
O nosso projeto pode ajudar em estudos para saber a relação de benefício em tornar os jogos como algo mais educativo
